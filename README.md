# 災害天氣預報 ｜ 問答小幫手

<img src=".github/img/IMG_8684.jpg" width=300>

資訊管理課程專案，目標幫助人們進行洪災防範與預警

### 功能：
- Line機器人預警
- 連接氣象局API
- 試算數值是否達警戒值
- 綁定地區播報
- 查詢即時觀測數據
- chatGPT問答功能
